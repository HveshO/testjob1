# Тестовое задание Python Junior +
## Версия Python - 3.9
Для установки:
```sh
git clone https://github.com/HveshO/testjob1.git
```
Далее по выбору:
-powershell commands:
```sh
venv\Scripts\activate
pip3 install -r requirements.txt
python app.py
venv\Scripts\deactivate
```
-Linux commands:
```sh
source activate venv
pip3 install -r requirements.txt
python3 app.py
deactivate
```
